# 🛡️ Henry Security - AI Log Analyzer

Ferramenta de Inteligência Artificial para análise de logs de segurança (Blue Team). 
Utiliza o modelo Gemini 2.5 Flash para detectar padrões de intrusão, SQL Injection e Brute Force.

## 🚀 Tecnologias
- Python 3.12
- Streamlit (Interface Web)
- Google GenAI SDK (Inteligência Artificial)

## 🛠️ Como usar
1. Faça o upload de um arquivo .txt ou .log.
2. Clique em "Executar Protocolo de Análise".
3. Receba o relatório técnico gerado pela IA.
